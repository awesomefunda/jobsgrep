# JobsGrep

Self-hosted job search web app. Type a natural language query, get a scored Excel tracker sourced from public APIs — Greenhouse, Lever, Ashby, HN Who's Hiring, YC companies, USAJobs, and more.

## Quick Start

```bash
cd jobsgrep
cp .env.example .env        # add GROQ_API_KEY at minimum
pip install -e .
python scripts/seed_companies.py   # optional: pre-seed 50 known company ATS slugs
jobsgrep serve               # → http://localhost:8080
```

Or with Docker (auto-validates config and generates tokens on first run):

```bash
cp .env.example .env        # edit as needed
docker compose up
```

## Environment Variables

| Variable | Required | Description |
|---|---|---|
| `GROQ_API_KEY` | **Yes** | LLM for query parsing and scoring fallback |
| `GEMINI_API_KEY` | Recommended | Primary scorer — 1,500 req/day free tier. Used first if set. |
| `JOBSGREP_MODE` | No (default: `LOCAL`) | `LOCAL` \| `PRIVATE` \| `PUBLIC` |
| `JOBSGREP_ACCESS_TOKEN` | PRIVATE mode | Bearer token. Auto-generated by Docker if not set. |
| `MIN_FIT_SCORE` | No (default: `0.7`) | Minimum score (0–1) to include a job in the report |
| `USAJOBS_API_KEY` | No | Register free at [developer.usajobs.gov](https://developer.usajobs.gov) |
| `ALLOW_SCRAPE` | No (default: `false`) | Force-enable JobSpy scraping in non-LOCAL modes |
| `PORT` | No (default: `8080`) | Server port |

See `.env.example` for the full list.

## Deployment Modes

Controlled by the `JOBSGREP_MODE` environment variable:

| Mode | Auth | Scraping | Caching | Use case |
|---|---|---|---|---|
| `LOCAL` | None | Allowed | 1 hour | Personal laptop use |
| `PRIVATE` | Bearer token | Disabled by default | 1 hour | Shared via Cloudflare Tunnel / ngrok |
| `PUBLIC` | IP rate limiting | Disabled | Disabled | VPS / cloud deploy |

Scraper sources (JobSpy) are **compile-time gated** — attempting to call them in `PUBLIC` or `PRIVATE` mode raises an error before any network call is made.

## LLM Providers

JobsGrep uses a unified provider chain. The first available key wins:

1. **Gemini 2.0 Flash** — set `GEMINI_API_KEY` (1,500 req/day free, recommended)
2. **Groq Llama 3.3 70b** — set `GROQ_API_KEY` (fast, generous free tier)
3. **Groq Llama 3.1 8b** — automatic fallback if 70b is rate-limited

Both keys are used for NL query parsing and job scoring via the same code path. Adding a new provider (OpenAI, Ollama, etc.) requires only a new function in `jobsgrep/llm.py`.

## Excel Report — Built-in Tracker

The generated `.xlsx` is designed to be your full job search tracker, not just a reference list.

**Sheet 1: Job Tracker**
- Left side (dark header): auto-filled by JobsGrep — rank, score, company, title, location, salary, URL, matching/missing skills, red flags
- Right side (teal header): your tracking columns — **Status** (dropdown), Applied Date, Outreach Sent, Response, Phone Screen, Interview, Offer, Notes
- Status pre-filled as `New` for every row — just change it as you progress
- Freeze panes keep Company + Title visible while scrolling into tracking columns

**Sheet 2: All Jobs Found** — complete reference list before scoring

**Sheet 3: Search Summary** — original query, parsed interpretation, sources searched, jobs per source

## Search History

Past searches are stored locally at `~/.jobsgrep/search_history.json` and shown on the homepage. Click any past query to re-run it. History is kept for the 50 most recent searches.

```
GET    /api/history    List past searches
DELETE /api/history    Clear history
```

## ATS Slug Discovery

Expand the list of companies searched by running the discovery script:

```bash
# Probe up to 500 YC hiring companies
python scripts/discover_companies.py --limit 500

# Also probe companies from your own CSV (requires a "name" column)
python scripts/discover_companies.py --csv my_companies.csv

# Seed 50 well-known companies instantly (no network calls needed)
python scripts/seed_companies.py
```

Mappings are cached in `~/.jobsgrep/company_ats_mapping.json` and never removed on re-runs — only new slugs are added.

## API Endpoints

```
POST   /api/search        Start a search, returns task_id
GET    /api/status/{id}   Poll task status
GET    /api/stream/{id}   SSE progress stream
GET    /api/download/{id} Download Excel report
GET    /api/sources       List enabled sources with legal classification
GET    /api/history       List past searches
DELETE /api/history       Clear search history
GET    /api/health        Source API health check
GET    /docs              Interactive API docs (Swagger UI)
```

## CLI Commands

```bash
jobsgrep serve                                    # Start web server
jobsgrep search "Staff SDE remote Python"         # CLI search → Excel file
jobsgrep discover [--limit 500]                   # Probe YC companies for ATS slugs
jobsgrep sources                                  # List enabled sources for current mode
jobsgrep health                                   # Test all source APIs
jobsgrep add-company <name> <greenhouse|lever|ashby> <slug>
```

## Legal

This tool queries **public APIs only** (except JobSpy in LOCAL mode). Every request uses an honest `User-Agent` header. Source rate limits are respected with a 50% buffer. An append-only audit log is written to `~/.jobsgrep/audit.log`.

See [DISCLAIMER.md](DISCLAIMER.md) for full details on ToS compliance per source.
